## Installation
* Clone this project

* Create a virtualenv
```bash
python3 -m venv venv
```

* Install dependencies
```bash
make pip-install
```

* Install pre-commit
```bash
make install-pre-commit
```

* Run the command below
```shell
  $ pip install --editable .
```
